# Devil's Sight 


***Prerequisites*** Level 2

You can see normally in [Dim Light](dim-light-xphb.md) and [Darkness](0.%20DM%20Screen/Variant-rules/darkness-xphb.md)—both magical and nonmagical—within 120 feet of yourself.
