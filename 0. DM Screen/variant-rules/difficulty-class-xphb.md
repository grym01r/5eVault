# Difficulty Class
*Source: Player's Handbook (2024) p. 366. Available in the <span title='Systems Reference Document (5.2)'>SRD</span> and the Free Rules (2024)* 

A Difficulty Class (DC) is the target number for an ability check or a [saving throw](0.%20DM%20Screen/variant-rules/saving-throw-xphb.md).
