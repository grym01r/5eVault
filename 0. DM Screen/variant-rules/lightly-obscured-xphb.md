# Lightly Obscured
*Source: Player's Handbook (2024) p. 370. Available in the <span title='Systems Reference Document (5.2)'>SRD</span> and the Free Rules (2024)* 

You have [Disadvantage](0.%20DM%20Screen/variant-rules/disadvantage-xphb.md) on Wisdom ([Perception](0.%20DM%20Screen/skills.md#Perception)) checks to see something in a Lightly Obscured space.
