# Attack Roll
*Source: Player's Handbook (2024) p. 361. Available in the <span title='Systems Reference Document (5.2)'>SRD</span> and the Free Rules (2024)* 

An attack roll is a [D20 Test](0.%20DM%20Screen/variant-rules/d20-test-xphb.md) that represents making an attack with a weapon, an [Unarmed Strike](0.%20DM%20Screen/variant-rules/unarmed-strike-xphb.md), or a spell.
