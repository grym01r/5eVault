# Creature
*Source: Player's Handbook (2024) p. 364. Available in the <span title='Systems Reference Document (5.2)'>SRD</span> and the Free Rules (2024)* 

Any being in the game, including a [a player's character](0.%20DM%20Screen/variant-rules/player-character-xphb.md), is a creature.
