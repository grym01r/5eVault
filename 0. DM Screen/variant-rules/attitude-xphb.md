# Attitude
*Source: Player's Handbook (2024) p. 361. Available in the <span title='Systems Reference Document (5.2)'>SRD</span> and the Free Rules (2024)* 

A [monster](0.%20DM%20Screen/variant-rules/monster-xphb.md) has a starting attitude toward a [player character](0.%20DM%20Screen/variant-rules/player-character-xphb.md): [Friendly](0.%20DM%20Screen/variant-rules/friendly-attitude-xphb.md), [Hostile](0.%20DM%20Screen/variant-rules/hostile-attitude-xphb.md), or [Indifferent](0.%20DM%20Screen/variant-rules/indifferent-attitude-xphb.md).
