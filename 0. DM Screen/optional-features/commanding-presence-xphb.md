# Commanding Presence 


When you make a Charisma ([Intimidation](0.%20DM%20Screen/skills.md#Intimidation), [Performance](0.%20DM%20Screen/skills.md#Performance), or [Persuasion](0.%20DM%20Screen/skills.md#Persuasion)) check, you can expend one Superiority Die and add that die to the roll.
