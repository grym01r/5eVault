# One with Shadows 


***Prerequisites*** Level 5

While you're in an area of [Dim Light](0.%20DM%20Screen/variant-rules/dim-light-xphb.md) or [Darkness](0.%20DM%20Screen/variant-rules/darkness-xphb.md), you can cast [Invisibility](invisibility-xphb.md) on yourself without expending a spell slot.
